# Centr-Krasok AI Telegram Bot

Мощный интеллектуальный Telegram-бот для магазина «Центр Красок», построенный на базе собственной RAG (Retrieval-Augmented Generation) архитектуры. 

Бот умеет «читать» ассортимент магазина, понимать семантику запросов клиентов на естественном языке и моментально выдавать точные ответы с ценами, скидками и характеристиками.

---

## Технологический стек

- **Core:** Node.js, TypeScript, Telegraf (Bot API)
- **Vector Search (RAG):** Supabase (PostgreSQL + pgvector)
- **Local Machine Learning:** `@xenova/transformers`
- **Embedding Model:** `Xenova/paraphrase-multilingual-mpnet-base-v2` (локальная генерация векторов размерностью 768)
- **LLM Engine:** Google Gemini
- **Scraping Engine:** Cheerio, Axios

---

## Как работает?

1. **Scraping Pipeline:** `parser.ts` агрегирует тысячи товаров с сайта (названия, цены, описания) и складывает их в чистый текстовый формат.
2. **Local Vectorization:** Скрипт `vectorStore.ts` разбивает данные на смысловые чанки и прогоняет их через тяжелую ML-модель `mpnet-base-v2` прямо на CPU/GPU сервера (без вызовов сторонних API).
3. **Database Injection:** Полученные векторы (768d) заливаются в высокоскоростную базу Supabase с векторными индексами.
4. **Smart Retrieval:** При вопросе от юзера, бот векторизует его запрос, делает "similarity search" по Supabase и достает ТОП релевантных товаров.
5. **LLM Synthesis:** Найденные товары скармливаются нейросети, которая отвечает клиенту в корпоративном и вежливом стиле.

## Инфраструктура (Self-Hosted)

Проект работает на собственной выделенной инфраструктуре:

- **Self-Hosted Supabase:** Развернут на собственных серверах (Hetzner). Обеспечивает полный контроль над базой данных, отсутствие лимитов на количество запросов и абсолютную приватность векторных хранилищ.
- **On-Premise AI:** Векторизация (генерация эмбеддингов) выполняется локально на железе сервера, что исключает утечки корпоративных данных (каталога) через сторонние API и экономит деньги.
- **Управление и Деплой (Coolify):** Автоматизированное развертывание и управление контейнерами через Coolify. Обеспечивает бесшовный CI/CD процесс и легкое масштабирование.
- **Отказоустойчивость и Мониторинг:** В систему интегрирован бот-надсмотрщик, который непрерывно следит за здоровьем (health check) всех узлов инфраструктуры. В случае любых сбоев администраторы моментально получают уведомления для оперативного реагирования.
- **Автономность:** Единственный внешний вызов идет к LLM-провайдеру для финального форматирования текста. Вся тяжелая логика поиска, парсинга и расчетов жестко изолирована внутри нашего контура.
